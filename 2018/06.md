# Negative indexed arrays (current)

```php, [.highlight: 4, 6, 8]
$a = array_fill(-2, 3, true);

array(3) {
  [-2]=>
  bool(true)
  [0]=>
  bool(true)
  [1]=>
  bool(true)
}
```

# Negative indexed arrays (new)

```php, [.highlight: 4, 6, 8]
$a = array_fill(-2, 3, true);

array(3) {
  [-2]=>
  bool(true)
  [-1]=>
  bool(true)
  [0]=>
  bool(true)
}
```

# Is Countable

```php
// before 
if (is_array($foo) || $foo instanceof Countable) {
    return count($foo);
}

// after
if (is_countable($foo)) {
    return count($foo);
}
```
